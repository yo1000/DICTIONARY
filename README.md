# DICTIONARY
NAMING DICTIONARY

## Controller

| Object               | Naming    | Note            |
|:---------------------|:----------|:----------------|
| Class - Page         | *Page     | @Controller     |
| Class - API          | *Resource | @RestController |
| Method - HTTP.GET    | get*      |                 |
| Method - HTTP POST   | post*     |                 |
| Method - HTTP PUT    | put*      |                 |
| Method - HTTP DELETE | delete*   |                 |

## Service

| Object          | Naming      | Note        |
|:----------------|:------------|:------------|
| Class           | *Service    | @Service    |
| Method - Create | create*     |             |
| Method - Read   | find*       |             |
| Method - Update | update*     |             |
| Method - Delete | remove*     |             |

## Repository

| Object          | Naming      | Note        |
|:----------------|:------------|:------------|
